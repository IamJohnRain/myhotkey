# myhotkey

## 使用方法

1. windows下 双击exe文件运行即可 (免安装`autohotkey`)

2. 安装`autohotkey`，双击运行`.ahk`文件 (软件官网地址: https://www.autohotkey.com/)

## 快捷键说明

### 媒体控制

`PrintScreen` → 音量减小

`Scrolllock` → 音量增大

`Pause` → 音乐暂停

### 方向控制

`CapsLock + h` → 方向左

`CapsLock + j` → 方向下

`CapsLock + k` → 方向上

`CapsLock + l` → 方向右

`CapsLock + u` → 向上翻页

`CapsLock + i` → 光标到句首(home键)

`CapsLock + o` → 光标到句尾(end键)

`CapsLock + p` → 向下翻页
